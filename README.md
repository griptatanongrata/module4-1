//Data types & variables//

_Data Types_

-help us organize data
-Types:

1. String
2. Number
3. Boolean
4. Undefined
5. Null

-String: text (wrapped in code)
example: console.log("Hello World")
-The first index of a string starts at 0. To find the first index of a string, type console.log("Justin"[0]) to retrieve the letter "J".
-To retrieve the length(number of indexex) in a string, type console.log("...".length). The console will say 3.

-Number: any number, no quotes
example: console.log(5-1) -> output = 4

-Boolean: true, or false (essentially yes, or no)

-Null: an empty value you set on purpose

-Undefined: a value that was never set (variable not declared)

_Variables_

-stores data
-Types:

1. var (not utilized anymore)
2. let (industry standard)
3. const (alternative to standard/locks the value at creation [unable to change])

-example for utilization of variables in index.js

_Equality_

"==": checks the values of variables or data types
"===": checks the values and the types of variables or data types
"!=": checks to see if the values are not equal
-examples shown in index.js
